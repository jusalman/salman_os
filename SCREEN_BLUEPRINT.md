# SALMAN OS Screen Blueprint

## Purpose

이 문서는 SALMAN OS v1 화면 구성을 정의한다. v1은 내부 직원용 고객사 운영센터이며, 고객사별 자료, 업무, 일정, 비즈머니, 링크, 로그를 한 화면에서 연결해 보여준다.

## Screen List

### 1. Login Screen

입력 항목:

- 공통 비밀번호
- 본인 이름

기준:

- 공통 비밀번호는 `salman1!`이다.
- 직원별 이메일 로그인은 v1에서 제공하지 않는다.
- 외부 고객 포털 진입 버튼은 제공하지 않는다.

### 2. Client List Screen

고객사 목록을 보여준다.

표시 항목:

- 고객사명
- 담당자
- 상태
- 오늘 일정 수
- 미완료 업무 수
- 비즈머니 확인 필요 여부
- 최근 로그 시간

### 3. Client Operation Center

고객사 상세의 핵심 화면이다.

상단:

- 고객사명
- 담당자
- 상태
- Google Drive 루트 링크
- 최근 업데이트 시간

본문 패널:

- 자료
- 업무
- 일정
- 비즈머니
- 주요 링크
- 운영 로그

이 화면은 SALMAN OS v1의 중심이며, Google Workspace 대체 화면이 아니라 운영 정보를 연결하는 화면이다.

### 4. Files Panel

Google Drive 원본 파일 링크를 보여준다.

표시 항목:

- 파일명
- 파일 유형
- Drive 폴더 경로
- 상태
- 등록자
- 등록일
- 원본 열기

삭제 동작은 제공하지 않고 `99_Archive` 보관 처리를 제공한다.

### 5. Tasks Panel

고객사별 업무를 보여준다.

표시 항목:

- 업무명
- 상태
- 우선순위
- 담당자
- 마감일
- 관련 자료
- 메모

### 6. Events Panel

Supabase 기반 내부 일정을 보여준다.

표시 항목:

- 일정명
- 날짜
- 시작 시간
- 종료 시간
- 담당자
- 상태
- 메모

Google Calendar 연동 UI, 캘린더 연결 버튼, 고객사별 실제 Google Calendar 생성 기능은 제공하지 않는다.

### 7. Business Money Panel

비즈머니 확인 상태와 링크를 보여준다.

표시 항목:

- 항목명
- 확인 링크
- 상태
- 마지막 확인일
- 확인자
- 메모

v1에서는 자동 수집 상태처럼 보이는 UI를 만들지 않는다.

### 8. Smart Operation Views

템플릿 기반 운영 뷰를 보여준다.

예시:

- 오늘 확인할 항목
- 마감 임박 업무
- 확인 필요 비즈머니
- 최근 업로드 자료
- 최근 보관 처리 파일

AI 채팅, AI 추천, AI 직원 실행 버튼은 제공하지 않는다.

## Navigation

권장 흐름:

1. 로그인
2. 고객사 목록
3. 고객사 상세
4. 자료, 업무, 일정, 비즈머니, 링크, 로그 확인
5. 필요한 상태 업데이트

## v1 UI Exclusions

- OpenAI API 설정 화면
- AI 직원 실행 화면
- Playwright 자동화 화면
- Google Calendar 연결 화면
- 외부 고객 포털 화면
