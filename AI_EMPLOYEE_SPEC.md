# SALMAN OS AI Employee Spec

## Purpose

이 문서는 SALMAN OS의 AI 직원 기능을 v1에서 어떻게 다룰지 정의한다. 결론적으로 v1에서는 AI 직원 기능을 실행하지 않으며, v1.5 이후 활성화 예정 기능으로만 문서화한다.

## v1 Decision

SALMAN OS v1은 내부 직원용 MVP다. v1의 목표는 고객사별 자료, 업무, 일정, 비즈머니, 링크, 로그를 한 화면에서 연결해 보여주는 운영센터를 검증하는 것이다.

따라서 v1에서는 AI 직원이 업무를 판단하거나 실행하지 않는다. OpenAI API도 사용하지 않는다.

## Disabled in v1

다음 기능은 v1 범위에서 제외한다.

- OpenAI API 호출
- AI 직원 응답 생성
- AI 직원 업무 실행
- AI 기반 일정 생성
- AI 기반 파일 분류
- AI 기반 비즈머니 판단
- Playwright 자동화 실행
- 외부 서비스 자동 로그인 또는 자동 수집

## Allowed in v1

v1에서 허용되는 것은 AI 실행이 아니라 템플릿 기반 정리다.

- 고객사 상태 템플릿
- 업무 우선순위 템플릿
- 일정 요약 템플릿
- 비즈머니 확인 필요 템플릿
- 최근 로그 기반 운영 체크리스트

이 기능은 Smart Operation Views라는 이름으로 제공할 수 있지만, AI가 아니라 정해진 규칙과 화면 템플릿으로 구성한다.

## v1.5 Direction

AI 직원 기능은 v1.5 이후 다음 조건이 충족된 뒤 활성화한다.

- 운영 데이터 구조가 안정화되어 있다.
- 직원이 실제로 반복 확인하는 업무 패턴이 확인되어 있다.
- AI가 실행할 수 있는 업무와 실행하면 안 되는 업무가 분리되어 있다.
- 사용자 승인 없이 외부 시스템에 영향을 주지 않는 안전장치가 있다.

## Product Language

v1 문서와 화면에서는 "AI가 처리합니다" 같은 표현을 쓰지 않는다. 대신 "템플릿 기준으로 정리", "운영 상태 보기", "확인 필요 항목"처럼 내부 MVP에 맞는 표현을 사용한다.
